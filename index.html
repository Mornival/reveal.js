React

React, what is it ?

React is a JavaScript library for building user interfaces.

React using

React is used by web developers

Why choose react?

  • React seems to be the good tool for building scalable web applications.
  • React is relatively easy to learn, with a clear and concise syntax.

History of react

  • React was created by Jordan Valke in 2011.
  • React source code open May 2013
  • The first release of the library was released in March 2013(v 0.3.0). The current version is React v18.0.

React features

  • Virtual DOM
  • Component development
  • Using JSX and TSX
  • Declarative
  • React hooks

Virtual DOM

  • The entire structure of a web page can be represented using the DOM (Document Object Model)
  • To solve the performance problem, the concept of the virtual DOM was born.
  • The virtual DOM is a lightweight copy of the standard DOM.
